2021/6 - 合肥·桂林·江西行

2021·6月 合肥·桂林·江西行

时间：2021/06/08 - 06/24

总路线：合肥(2) -> 桂林(4) -> 岳阳(1) -> 武汉(2) -> 庐山(2) -> 景德镇(1.5) -> 三清山(3) -> 衢州(1.5) = 16

前期准备中…

三月份的时候，计划着想重逛合肥。列了大致计划，其中就包含了曾经点过9号和27号的良子足浴。心血来潮想看看涨没涨价，发现美团无法搜索到。hhh这能难倒我？去过四次的我现在闭上眼睛都能把那个当时还在修路的十字路口描绘出来。打开百度地图 -> 我的收藏，不错，位置还在。点开了全景地图，重走了一下合肥。嗯…店怎么没了？我甚至无法找到那个十字路口… 后来还是记忆力过人的老谈帮我回忆，找到了那个十字路口，看到了旁边的那个眼镜店。用了百度地图的时光机回溯功能，可以看到18年好像还在，现在已经没有了…

今天准备开始做详细计划了，很想去再吃一次竹寿司的午市套餐，不到30的拉面还有咸香可口的火炙三文鱼。结果店又没了？哎…父母来的时候我们还一起吃过的。

又想起了我很多个上午，放弃了学校附近的几个影城，骑车40min到南边一个小角落看上午特价9.9的电影。电影院名字已经记不住了，不过我记得当年下了一个app。翻了翻，从已购中找到了「梦露影院」。没错，就是他！打开之后，「因疫情原因，本店暂时不开放」。😭 这次的回忆之旅看来注定要受挫。

得，19.8的富川日式料理也没了…

日程安排

合肥篇

6/8

早起后打车去机场，西北路小堵，还好提前了一点出发。碰到了一个头发发白的司机，几乎每次停车都拉手刹。我有点没看懂…大叔也不主动搭话，我也找不到合适的话题，于是车上唯一的娱乐活动就变成了：期待每一个红灯，然后猜测大叔这次会不会拉手刹 -.-

飞机的一路很顺利，虽然起飞前说飞机故障了，不过很快就修好了。在值机选座的时候，全飞机都选满了，只剩一个座49B…?怎么回事，大家都到了就等我了？上飞机的时候我记错了，看到48B上做了一个小孩子，我抛出了微笑版质疑。还好尴尬只持续了10s… 下飞机弄出安康码，安徽的码稍微严格一下，需要身份证的起止日期。出了门，到了熟悉地方买大巴票，一号线到三孝口，四号线到皇冠假日。刚上车车就出发了。不过小姐姐告诉我一号线可以在大西门下，这样近一点，确实。

大西门就位后，开始想办法骑车到良子足浴。发现路旁一排电动的美团和青桔。wow！很酷啊，共享电动车有点东西。并且还为这个共享电动车配了脚蹬子。开在路上，司机坐着休息，脚蹬子在旁边自己转的不亦乐乎，有点可爱。并且合肥的美团共享单车好新啊！是刚换的吗？车座很高，可以自己调，并且很明显这些车们都没有经过岁月和下雨的洗礼，显得有些青涩。

良子足浴高登店是个老店了，那年我去另一个良子的时候这个店就在了。不过老谈听说除了我们去的那个店，其他的不正宗…emmm A说B不正宗既不能说明A正宗，也不能说明B正宗，只能说明A和B不是一宗。提前在美团上选了「良子养生」并查好了攻略，6 > 18 > 9号。去了之后6号正好在，她是88年的姐姐。为了和她互动以及享受完全的按摩体验，我放弃了滚子局… 她也确实挺能讲的，我们有来有回，前期我多说一点，后期聊到婚姻那就是她的主场了。她自称自己为大姐，还主动聊到店里最年轻的只有19岁。中途她说：

她：「很多客人看你是大姐，虽然按得还行，但是人家还是喜欢找年轻的」

我：「年轻的可能按摩水平不太行」

她：「那不重要，按得是一方面，好看是一方面。你们都是这么想的吧」

我：「…」

出门骑车，找到了老地方的卡旺卡，点了一杯经典的「黑全套」。看到「王小哥哥」还是有一点傻眼的…

去之前就在担心学校会不会不让进，去了西门之后，迎面而来那个熟悉的门卫小哥。小哥露出冷酷而帅气的表情，拒绝和我互动。往右面一指，暗示我他这个是出口，要从入口进。入口是两个陌生的大爷，我一想，这完了。不出所料，果然被大爷拦住了：

我掏出我的学生证

他：毕业生也不行。

我搬出「旅途遥远，来都来了」卖惨

他掏出「疫情当前，大局为重」把球踢了回来

我表示我很健康

他表示不行，特别是留学生，肯定不让进。（？我脸上写了英文吗…）

我：我约了老师和同学一起吃个饭

他：那你让你同学出来，然后oeffoiewbfufsandw^&%$%&@ (合肥方言降维打击)

我：？？？好吧… 打开微信语音「喂，要不你出来接我一下，或者我看看其他门能不能进」

他露出轻蔑一笑，内心os:「其他门肯定也不行」

打电话跟老张求助，她说可以试试小西门，因为饭点人多，可能不会卡门。徒步走到小西门，我瞬间发现两个问题：左右各一个大爷。我瞅了一眼，这大门怎么关的，诶那里面那两个人是怎么进去的？大脑疯狂旋转…原来是读者服务部！我（看起来）充满信心的打开读者服务部的门，走到门禁，看着敞开的大门，装着忽视左右两位大爷的深情走了进去，呼！进去的那一刻感觉天空无比明亮。

后来的几天，发现确实饭点的时候很好进，老张也把她的学生卡给了我，于是门禁的问题不再存在。

进了校园之后当然是各种怀旧，抓了一个共享单车，先去吃个饭吧！来到了日思夜想的茶餐厅，看到菜单我惊了一下，这居然还能降价？原来最高级套餐好像是17？现在的学生好幸福啊…借了一个陌生学弟的卡来刷，生疏的我居然连餐具也没找到，有些狼狈…

饭还是很好吃的！不过这饭量也太大了…我以前这么能吃吗？

饭后7点了，730就要天黑，怀旧也得抓紧时间。骑上美团小车，开始打卡拍照。每一处都有着无数的回忆，孑然一人，既有即将溢出来的快乐，也有一种四下无人分享的遗憾。

不知不觉天黑了，我也骑到了东区操场旁边，想到以前晚上的跑步时光，诶！走两圈吧。于是我放下手里的西瓜和书包，开始活动脚踝，寻找机会，逆时针merge进了人群。跑起来之后：

我感觉自己又是那个年轻的少年了。脚下的步伐还是那么轻盈，心想着今天也要跑五圈。

跑着跑着，感觉有些累，哎，要不然跑一圈，冲一圈吧

哎，这个小腿好酸，之后还有两周的活动呢，要不算了吧

那晚在东区休息的人可能会看到，一个男孩一脸自信的踏上操场，热身两分钟，跑了一圈，然后背上书包走了…

经过中区的隧道，和老张碰头。决定步行去五彩城，那是曾一起看「爱乐之城」的地方。可惜当年那个「尴尬的情人节电梯」现在关了。这告诉我有时候回到老地方，不一定能重现那时的心情。还有的时候，连老地方都没得看。

一时兴起，去KTV小唱。经过一段时间的训练，我现在从费劲唱E到了费劲唱升F。看来还是要坚持训练啊，一段时间我大概不会再去唱歌了。

回到锦江之星之后，发生了没有房的小插曲，不过最后顺利解决。看来买的金卡会员还是可以在无房的时候得到免费升级的。不过合工大的这个锦江实在是离的太远了，下次还是定学校附近的比较好，能多睡十分钟。回来看自己的骑行记录，一共骑了2h。感觉还很有力量，要开始拉练的生活了。

6/9

晚上大概是因为卡旺卡，再次获得失眠。于是我未雨绸缪的删除了晚上熬夜海底捞的计划，准备踏踏实实躺下睡个好觉，事实证明，连续熬夜后，想在调回来作息是一件很难的事。这一周又是学习「睡眠重要性」的一周。

打卡肥叔锅贴，脆脆的很好吃，下次来合肥还要来吃。营养沙汤一大碗，阿姨先打了个生鸡蛋，然后把汤倒进碗里，形成了一晚蛋花汤，很有趣。味道也很不错。烤肠也很好，都是以前的味道。如果说铁板炒饭是校内怀念top1，那么肥叔就是校外怀念top1。

今天继续了打卡拍照，西区被我转了一圈后，和老张约了《寂静之地2》。总体上来讲没有1那么出彩，也没有1那么恐怖。很少有「突然吓人」的情节，这是一大进步，不过也少了一些「细思极恐」的桥段，还需要努力。出来后走在天鹅湖边，想到了之前的王冬青毛概作业的Proj：二十岁的胖子

旁边就是安徽电视台，有着我奶奶最爱看的「男生女生向前冲」以前是能看到整个的跑道的，现在好像在拆了重盖，有点遗憾没能拍到。我们沿着天鹅湖+绿轴公园狂走，跟老张分享了我回国的艰辛之旅。晚饭收于晋家门，终于吃到了这个店，感觉一般般。肉串不香，倒是蜜桃冰粉很令人眼前一亮，我可以一人吃两碗，不需要给我别的了。

合肥君悦一直不开放5K兑换，让我错失Q2的活动，正好我想俯视一下合肥，于是我们一起上了君悦的40F lobby。不得不说，虽然隔得那么远，还是有点恐高的。这也是第一次没房卡来这边酒店，很是怕有人和我互动hhh

合肥有一条我喜欢的路叫「休宁路」，我经常选择在夜晚骑行这条路。因为他没有红绿灯 + 回学校的这一个方向全是下坡。很多次的忧愁思绪，都在这条路的晚风中化解的无影无踪。这次打车来的时候，大叔很巧的选了这条路。我们在晚上回学校的时候也重走了这条路。合肥的电动车还是那么多，疯狂在旁边超车，搞得我有点怕…

民航大厦下之后先北上去「阿秀米粉」

  • 然后往东走到漓江边上，再沿着漓江边上往北走到宾馆。

米粉怎么吃：

  • 首先桂林米粉是按两卖的，一两二两三两，凡是看到哪家店用大份小份大碗小碗或者直接套餐来卖的话你可以走出去了。
  • 米粉按形状分两种，圆粉和切粉，点单的时候要跟老板说要哪种粉，不说的话默认给圆粉。两种粉口感不同，建议都试一下，可以各点一两（我个人比较喜欢切粉）
  • 传统的桂林米粉是干拌型的，本地人叫卤菜粉或者卤粉，吃的时候完全没有汤，新鲜米粉在热水里烫一下捞出来放碗里，然后淋上特制的卤水和明油，再放上切得薄薄的肉片
  • 所有肉类都是现切的，有卤马肉、卤猪肉、卤牛肉、广式香肠、叉烧、锅烧等肉类可以选，下单后你需要站在窗口看着阿姨给你切肉，**很多没吃过米粉的人以为下单之后在座位上等着就可以了，这一看就是外地人。。。**本地人会站在窗口像食堂打菜一样看着阿姨切肉，告诉她你要什么肉，可以选一种或多种。比如我这份加的是锅烧，卤肉和叉烧，注意，阿姨从窗口做好的这个样子的粉只能算半成品，你还要自己去加调料，有些人从窗口直接这样端走就坐下吃的，这一看又是外地人。。。。
  • 顺便说一下，所有肉菜里锅烧的地位是最高的，锅烧是用半肥半瘦的带皮猪肉先卤后炸到金黄酥脆，有厚厚的脆壳，所以又叫脆皮，桂林人检验一个米粉店的水平，一看卤水，二看锅烧
  • 一般每个米粉店都有一个桶，装有免费骨汤，很多本地人吃米粉的习惯是吃到最后再去盛一点骨汤，就着碗里的剩余调料喝汤。也可以吃到一半加汤，把卤粉变成汤粉

小逛老寨山公园？【好像不需要门票】

  • 路过兴坪汽车站，先买去阳朔的车票
  • 半小时上山，半小时下山

老寨山尽管路途陡峭，依然吸引了不少摄影发烧友奋起攀登。老寨山海拔300余米，虽不高，却有一千一百多个阶梯。见到圆拱山门走上圆拱山门过后，接近顶处会见两架铁梯，顺着铁梯往上就能到达山顶。千万别走错路。在山顶远眺，漓江在山脚下拐了个弯，一艘艘游船穿梭而过，远处山峦叠嶂，分外妖娆。尤其傍晚时分，对岸那几经沧桑形成的“大河背”数千亩冲积小平原，连绵山峦在夕照中一片微红，微红中透出点点绿意，这便是著名画家徐悲鸿盛赞的“兴坪夕照”。

老寨海拔300余米，虽不高但是爬起来很困难，因为没有正规的山路，台阶也是时有时无，时宽时窄。

之后从兴坪汽车站坐车到阳朔。逛西街。【这个票现场买】
